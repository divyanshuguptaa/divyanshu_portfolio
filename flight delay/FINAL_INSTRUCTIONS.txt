TROUBLESHOOTING:

Problem: "python is not recognized..."
Solution: Python is not installed or not in PATH. Install Python 3.11+

Problem: "No module named 'pandas'"
Solution: Run: pip install -r requirements.txt

Problem: Dashboard shows errors
Solution: Make sure you're in the correct directory and CSV file exists

Problem: Browser shows "Can't reach this page"
Solution: Dashboard isn't running yet. Wait 10 more seconds and refresh.

Problem: Want to stop dashboard
Solution: Go to the console window and press Ctrl+C
